# Orders system

## Структура и задание

Система поделена на следующие модули:
- `infrastructure.go`: Имитация работы с БД и внешними сервисами.
- `main.go`: Точка входа в приложение.
- `models.go`: Модели данных системы.
- `processor.go`: Основная бизнес-логика.
- `staff.go`: Система управления персоналом склада.

Почти в каждом модуле есть архитектурные ошибки, которые нужно исправить.



## Использование
Запуск системы:
```
go run .
```
